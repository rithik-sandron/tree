# Print your directory as a tree with colors.
- run `./tree <path>`
- running `./tree` without a **path** prints current directory `.`
  
__output:__
```console

./path
├── .DS_Store
├── sub-folder
│   ├── code.rs
│   └── readme.md
├── z-last.html
├── second.html
└── first.html
```
